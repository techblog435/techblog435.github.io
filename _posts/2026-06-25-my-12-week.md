## Exhibition Day Anticipation: Prepping components and dealing with faulty jumper wires

As the Digital Logic Design (DLD) project exhibition approaches, the lab environment turns electric. Preparing hardware models for display means translating abstract circuit schematics into physical prototypes. The process is never completely smooth—hours are spent tracking loose connections, replacing faulty breadboards, and swapping out deceptive jumper wires that look perfectly fine but fail to carry a signal. It is a rigorous exercise in hardware debugging, where patience and systematic troubleshooting are just as vital as the circuit design itself.

## Digital Logic Design (DLD): Designing combinational and sequential circuits

At the foundation of computing lies Digital Logic Design, the art of controlling electrical paths to perform mathematical operations. Throughout the semester, we transitioned from basic combinational logic—where outputs depend purely on current inputs using multiplexers and logic gates—into complex sequential circuits. Sequential design introduces memory, leveraging clock signals, flip-flops, and binary counters to track state changes over time, forming the actual functional basis of processing units.

## The Digital Ludo Dice: Building a randomized electronic dice on a 7-segment display

Our central project for the exhibition is a functional Digital Ludo Dice. The system replaces a standard plastic dice with a randomized electronic circuit that cycles rapidly through numbers 1 to 6 when a button is pressed. 



The internal architecture relies on a high-frequency clock generator feeding a binary counter, which is restricted to a modulo-6 sequence. To display the current state, a combinational decoder translates the binary output into driving currents for a standard 7-segment display, illuminating the correct LED segments ($a$ through $g$) to visually represent the rolled number instantly.

```csharp
// High-level logical representation of the Mod-6 Counter to 7-Segment Mapping
public class LudoDiceLogic
{
    public static int CounterStateState(int clockPulses)
    {
        return (clockPulses % 6) + 1;
    }

    public static string GetSevenSegmentOutput(int diceValue)
    {
        switch (diceValue)
        {
            case 1: return "0110000"; // Segments b, c active
            case 2: return "1101101"; // Segments a, b, d, e, g active
            case 3: return "1111001"; // Segments a, b, c, d, g active
            case 4: return "0110011"; // Segments b, c, f, g active
            case 5: return "1011011"; // Segments a, c, d, f, g active
            case 6: return "1011111"; // Segments a, c, d, e, f, g active
            default: return "0000000";
        }
    }
}
