## Database Foundations: Introduction to relational databases, schemas, and data integrity

At the heart of this semester is the deep dive into **Database Management Systems (DBMS)**. A relational database organizes data into points with defined relationships, structured neatly into tables. Managing this requires a solid understanding of schemas—the formal structure that defines how data is organized—and data integrity rules, which ensure information remains accurate and consistent over time.

```sql
-- Creating a basic student database schema with data integrity constraints
CREATE TABLE Students (
    StudentID INT PRIMARY KEY,
    FullName VARCHAR(100) NOT NULL,
    RollNumber VARCHAR(20) UNIQUE,
    EnrolledSemester INT CHECK (EnrolledSemester >= 1)
);

CREATE TABLE CourseEnrollments (
    EnrollmentID INT PRIMARY KEY,
    StudentID INT,
    CourseName VARCHAR(50),
    FOREIGN KEY (StudentID) REFERENCES Students(StudentID)
);

```
